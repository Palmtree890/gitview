GitHub Repo Explorer & Downloader Pro
A clean, modern web application that allows users to explore GitHub repositories, browse file structures in real-time, and download raw files directly from the raw.githubusercontent.com domain.

Two Search Modes: Search by GitHub username to see all public repos or search by keywords to find specific projects.
Interactive File Tree: Browse through repository folders and files without leaving the page.
Live Previews:
Syntax-highlighted code previews for JavaScript, Python, HTML, CSS, and more.
Image previews for popular formats (PNG, JPG, SVG, WebP).
Automatic README.md rendering for every repository.
Raw Downloads: A dedicated button that fetches files directly from raw.githubusercontent.com to get the unformatted source code.
Responsive Design: Optimized for both desktop and mobile viewing with a sleek GitHub-inspired dark theme.
